## 💻 Jak użyć w kodzie

```typescript
// Znajdź obiekt na scenie
const myObject = this.findActorByName('NazwaObiektu');
const meshComponent = myObject?.findComponentByClass(ENGINE.MeshComponent);

// Zastosuj materiał przezroczysty zielony:
meshComponent?.setMeshMaterial('@project/assets/materials/M_Transparent_Light_Green.material.json');

// Zastosuj materiał przezroczysty czerwony:
meshComponent?.setMeshMaterial('@project/assets/materials/M_Transparent_Light_Red.material.json');

// Lub inne przezroczyste:
meshComponent?.setMeshMaterial('@project/assets/materials/M_Transparent_Emerald.material.json');
meshComponent?.setMeshMaterial('@project/assets/materials/M_Transparent_Ruby_Red.material.json');
```

## ✨ Cechy materiałów

**Półprzezroczyste materiały (niebieskie/zielone/czerwone):**
- ✅ Pełny PBR workflow
- ✅ **Przezroczyste** (transparent: true, opacity 0.35-0.5)
- ✅ **Szklisty efekt** (niska metalness 0.1-0.3)
- ✅ **Błyszczące, gładkie** (roughness 0.1-0.2)
- ✅ **Silne odbicia** (envMapIntensity 1.5-1.8)
- ✅ **3 kolory x 3 odcienie** = 9 przezroczystych materiałów
- ✅ Idealne dla: szkła kolorowego, klejnotów, efektów świetlnych, wody kolorowej
